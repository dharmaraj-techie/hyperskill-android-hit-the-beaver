Description

You are going to visually spice things up in our game with five images. First of all, you will add an active image, which will let the user know they should click on the image. Then, you will add an inactive image which will let the user know the image is not currently active and that clicking on it won't do anything. Finally, a successful image will be displayed each time the user successfully clicks on the active image. 

You want the images you added to the project to behave like buttons. We can do that by adding an ImageButton element. You will set all the buttons to inactive at first. 

Objectives

  1. Add action to start new Activity. When the user clicks on the Next button, the game screen will now appear. 
  2. The name of the game screen activity must be GameScreenActivity
  3. Add five ImageButton . You can download all images for your project game here.

User interface should now have images on both the top and bottom of our screen.
