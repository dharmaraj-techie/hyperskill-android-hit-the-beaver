Description

It's time to build the foundations of a hit-the-beaver game by adding parts of the user interface and interacting with it through code. In this stage, your task is to create a simple application with Android UI widgets. You will begin in activity_main.xml, and add the buttons as the starting point of our game. 

For now, you will continue developing user interface by adding a place where the user can type in their name. This will come in handy later for displaying things like personalized messages or high-scores. You will use an input area to do that. 

Right now you need to hide the Start button after the user clicks it as in the example below.  

Objectives

  1. Add button Start with id start_button, Next button with id next_button and an EditText with id edit_text
  2. After pressing the Start button, will appear EditText and Next button.
  3. The Input type of the EditText must of type text (i.e) android:inputType="text"
