Description

At this stage, we will check the correctness of the input. The user should not enter a too long nickname to make it easier for others to recognize it. Therefore, you can add a restriction and a notification so that the user understands that he entered something wrong.

If the user enters a value that you don't want to accept, you can show a pop-up Toast that displays an error. 

Objectives

  1. Add Toast which says "Enter a valid name" when the user enters incorrect value. The value must be text only and must be between length 1 and 9.
  2. Add a function to begin Game, if the user enters a valid name. the name of the function must be beginGame. when that function gets called it should show a Toast saying "Begin Game". for testing please make the method public
